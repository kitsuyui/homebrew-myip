# homebrew-myip

Homebrew Installation of [go-myip](https://github.com/kitsuyui/go-myip).

## Installation

```console
$ brew tap kitsuyui/homebrew-myip
$ brew install myip
```

## Update

```console
$ ./generate.sh v0.2.1
```
